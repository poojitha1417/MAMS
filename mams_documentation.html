MAMS

Military Asset Management System

1. Project Overview

MAMS is a secure, digital ledger system for tracking high-value military assets. It ensures accountability across multiple bases by tracking every movement, from purchase to expenditure.

Assumptions: Ledger-based inventory calculation, Stateless JWT Auth, and Reference-based auditing.

Limitations: MVP lacks automated stock-out validation; requires manual entry for asset categorization.

2. Tech Stack & Architecture

Architecture: Client-Server REST API using a modular Express backend and a React-powered frontend.

LayerTechnology
FrontendReact 18, Vite, Tailwind CSS
BackendNode.js, Express 5.2
Database/ORMPrisma, SQLite (Dev) / PostgreSQL (Prod)
AuthenticationJWT (JSON Web Tokens), Bcrypt.js

3. Data Models / Schema

The system is built on four core entities:

  • User: Credentials and role assignment (ADMIN, COMMANDER, LOGISTICS).
  • Base: Physical installation records.
  • Asset: Definitions of equipment types.
  • Transaction: The immutable ledger entries tracking all movements.

4. Role-Based Access Control (RBAC)

Access is enforced via custom Express middleware: authMiddleware and roleMiddleware.

RolePermissions
ADMINFull System Access (User & Base Management)
COMMANDERBase-level Operations & Assignments
LOGISTICSSupply Chain & Purchase Records

5. Core Features Walkthrough

📊 Mission Dashboard

Real-time analytics engine providing closing balances, net movements, and visual asset distribution charts.

👥 Personnel & Command

Infrastructure management for establishing deployment bases and authorizing military personnel across roles.

🛡️ Logistics Operations

Strategic ledger for recording purchases, inter-base transfers, and field assignments/expenditures.
