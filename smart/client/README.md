# 🛡️ M.A.M.S - Military Asset Management System

A mission-critical dashboard for real-time tracking of tactical assets across global deployment bases. Featuring a high-contrast **Dark Tactical UI** designed for extreme visibility and rapid intelligence assessment.

## 🌗 Tactical Theme
- **Color Palette**: Slate 900 (Primary Ops), Slate 800 (Protocol Cards), Tactical Yellow (Primary Accent).
- **Core Stack**: React + Vite, Tailwind CSS v4, Lucide Icons, Recharts Analytics.

## 🚀 Deployment Protocols

### 1. Intelligence Backend
Navigate to the `server` directory and initiate the service:
```bash
cd server
npm install
npm run dev
```
*Environment variables should be configured in `.env` (Prisma/SQLite).*

### 2. Tactical Frontend
Navigate to the `client` directory and launch the command center:
```bash
cd client
npm install
npm run dev
```
